Summary: Latitude vs. X

This html webpage serves as a way to practice my abilities to create an html webpage and explore CSS and Boostrap. I use data and images that I used for a prior project where I examined the weather using a Weather API. In that project, I looked at how latitude effected the weather in various ways which you can see from the following visualizations and analysis. These include how latitude effects temperature, the level of cloud coverage, humidity and wind speed.

This page serves as my landing page or my "home" page. You can use the navbar to navigate to further analysis or simply click on one of the vizualations to the right of this text. I made these vizualizations in a Jupyter Notebook using MatPlotLib scatterplots. Some of the findings were not too unusual or unexpected such as the finding that as you move closer to the equator the max temperature of the area (latitude) increases.
